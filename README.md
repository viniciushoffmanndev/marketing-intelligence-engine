# Marketing Intelligence Engine

Plataforma de análise de dados orientada por Inteligência Artificial, construída com Python, Streamlit e OpenAI.

Este projeto transforma dados brutos de marketing em insights visuais dinâmicos utilizando linguagem natural, permitindo que usuários interajam com os dados como se estivessem conversando com um Analista de Dados.

## Visão Geral

O Marketing Intelligence Engine é uma aplicação interativa de analytics inspirada em plataformas modernas de BI conversacional orientadas por IA, como a Looqbox.

A aplicação permite:

- Carregar e analisar bases de dados de marketing
- Gerar visualizações dinâmicas automaticamente
- Solicitar insights em linguagem natural
- Realizar análises exploratórias
- Criar gráficos utilizando IA
- Interagir com dados através de um agente inteligente

A IA gera código Python dinamicamente com base na solicitação do usuário e executa a análise diretamente na interface do Streamlit.

---

## Funcionalidades

- Análises em linguagem natural
- Geração automática de gráficos
- Visualizações dinâmicas
- Análises exploratórias automatizadas
- Interface conversacional com Streamlit
- Geração de insights em tempo real
- Integração com OpenAI
